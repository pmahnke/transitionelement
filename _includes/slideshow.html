{%- comment -%} This include generates a dynamic slideshow from images defined in the page's front matter. It uses simplified IDs (img-1, img-2, etc.) assuming uniqueness per page. Cloudinary integration is enabled only in the 'production' environment. In other environments (e.g., 'development'), it uses the url directly. If there is only one image in slideshow_images, it outputs a plain tag. Usage: {%- include slideshow.html page=page -%} Parameters: - page: (Required) The current page object (e.g., `page` in a post/page layout). It expects `page.slideshow_images` to be an array of image objects in the front matter. Page Front Matter Structure (e.g., in your post.md): --- layout: post title: My Post slideshow_images: - url: "https://external.com/image1.jpg" # Full external URL alt: "Description for image 1" - url: "/assets/images/my-local-image.jpg" # Local path alt: "Description for local image" --- Cloudinary Integration: - Requires `site.cloudinary_cloud_name` and `site.url` to be set in your _config.yml for production builds. - Transformations used: `q_auto:best` (auto quality, best setting), `f_auto` (auto format). You can customize these or add more transformations (e.g., `w_800,h_450,c_fill`) by adding them to the `cloudinary_transformations` variable. {%- endcomment -%} {%- assign current_page = include.page -%} {%- assign images = current_page.slideshow_images -%} {%- if images and images.size > 0 -%} {%- assign num_images = images.size -%} {%- assign cloudinary_cloud_name = site.cloudinary_cloud_name -%} {%- assign cloudinary_transformations = "q_auto:best,f_auto" -%} {%- if num_images == 1 -%} {%- comment -%} Output a single image link {%- endcomment -%} {%- assign single_image = images.first -%} {%- comment -%} Determine the final image source for the single image {%- endcomment -%} {%- assign final_image_src = single_image.url -%} {%- if jekyll.environment == "production" and cloudinary_cloud_name -%} {%- assign image_source_for_cloudinary = "" -%} {%- if single_image.url contains "/" and single_image.url | slice: 0 == "/" -%} {%- assign image_source_for_cloudinary = site.url | append: single_image.url -%} {%- else -%} {%- assign image_source_for_cloudinary = single_image.url -%} {%- endif -%} {%- assign final_image_src = "https://res.cloudinary.com/" | append: cloudinary_cloud_name | append: "/image/fetch/" | append: cloudinary_transformations | append: "/" | append: image_source_for_cloudinary -%} {%- endif -%}
